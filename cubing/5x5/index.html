~elis/cubing/5x5/

This page contains only parity cases for the 5x5x5 and bigger cubes.

Edge Parity

Edge parity occurs on 4x4x4 (and larger cubes) when you have a single pair of edges that need to be swapped. This algorithm fixes the edge parity issue that can occur during the edge pairing stage.

Notation

(r2 B2 U2 l U2 r') (U2 r U2 F2 r F2 l') (B2 r2)
