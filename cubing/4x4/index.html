~elis/cubing/4x4/

This page contains only parity cases for the 4x4x4 and bigger cubes.

Parity for 4x4 and other even cubes

This parity algorithm is used to flip the entire edge between top front and top back side. This can only occur on even-layered cubes and works on any size of even layered cube.

Notation

(r2 R2 U2 r2 R2 u2 r2 R2 u2) U2

Parity for 4x4 and bigger cubes (even or odd)

This parity algorithm is used to resolve the single edge flip when two edges next to each other needs to switch places. This can occur on 4x4x4 and bigger cubes, both even and odd layered cubes.

Notation

(r2 B2 U2 l U2 r') (U2 r U2 F2 r F2 l') (B2 r2)
